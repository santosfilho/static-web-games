# Desafios de Desenvolvimento com IA — Microsoft Copilot Adventures

Este repositório contém uma coleção de pequenos laboratórios e desafios de desenvolvimento voltados para o uso prático de Inteligência Artificial (GitHub Copilot), baseados nos materiais da **Microsoft Copilot Adventures**.

## 🚀 O Projeto

A ideia principal é explorar como o GitHub Copilot pode auxiliar na criação de algoritmos, refatoração de código, escrita de testes e desenvolvimento de interfaces web interativas. Cada pasta contém um "laboratório" ou "aventura" específica com objetivos distintos.

## 📂 Estrutura de Labs

Atualmente, o projeto conta com:

### 🎮 [Warmup Adventure: Echo Chamber (Câmara do Eco)](./warmup-adventure/CopilotAdventures/echo-chamber/)
Um motor de análise de sequências numéricas com uma interface web temática inspirada em clássicos góticos.
- **Tecnologias:** JavaScript (Vanilla), HTML5, CSS3, Node.js.
- **Desafio:** Implementar lógica para detecção de Progressões Aritméticas (PA), Geométricas (PG) e Sequências Polinomiais.
- **IA Focus:** Geração de algoritmos matemáticos complexos e suítes de testes automatizados.

## 🛠️ Como Começar

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/seu-usuario/static-web-games.git
   cd static-web-games
   ```

2. **Explore os Labs:**
   Entre na pasta do desafio desejado e siga as instruções do `README.md` local. For exemplo:
   ```bash
   cd warmup-adventure/CopilotAdventures/echo-chamber
   # Veja o README local para instruções de execução e testes
   ```
